Three Assessment Pillars

Every audit evaluates your data center across three dimensions, producing a weighted overall score and letter grade.

⚡
Box 1 · 40% weight

Current Operations

PUE efficiency, server utilization, energy cost analysis, cooling performance, availability metrics, and carbon intensity scoring.

🗑
Box 2 · 30% weight

Legacy & Waste

Zombie server detection, over-provisioned resources, legacy hardware identification, cooling waste analysis, and stranded capacity.

🚀
Box 3 · 30% weight

Future Readiness

Capacity forecasting, hardware refresh planning, workload scheduling optimization, renewable energy adoption, and efficiency trends.

Quick Start

Up and running in under a minute.

# Install
$ pip install energy-audit

# Run a full audit
$ energy-audit run -p medium_enterprise -s 42

# Export PDF report
$ energy-audit run -p large_hyperscale --export-pdf report.pdf

# Compact dashboard
$ energy-audit dashboard -p medium_enterprise

# Interactive maturity assessment
$ energy-audit assess
